JackAces

HOME CHAT ABOUT US PROFILE

JackAces is a Web-Based Chat Server, created using WebSocket. It is tailored to IT users, with a wide range of chatrooms available to users, such as Gaming, Coding and Support.


JackAces features a wide range of integrated features. It allows users to create an account before beginning to chat online. It also includes file uploads, character limits for messaging, and a feature that saves chats between sessions.


JackAces was created for the 2026 First Year Computer Science Hackathon, as part of the CT101 Computing Systems module.
